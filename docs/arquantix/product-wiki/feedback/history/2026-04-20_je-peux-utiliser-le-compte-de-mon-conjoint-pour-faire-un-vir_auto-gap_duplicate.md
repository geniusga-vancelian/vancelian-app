# Je peux utiliser le compte de mon conjoint pour faire un virement vers mon compte Vancelian ?

## Client question
> Je peux utiliser le compte de mon conjoint pour faire un virement vers mon compte Vancelian ?

## Bot answer
> Non, vous ne pouvez pas utiliser le compte bancaire de votre conjoint pour faire un virement vers votre compte Vancelian.
> 
> Vancelian applique une règle stricte de **correspondance des noms** : le titulaire du compte bancaire émetteur doit correspondre au titulaire du compte Vancelian destinataire. Cette exigence découle des obligations de conformité AML/CFT (lutte contre le blanchiment et le financement du terrorisme) — Vancelian doit vérifier que les fonds proviennent bien de la personne qui les reçoit.
> 
> Si vous tentez un virement depuis le compte de votre conjoint vers votre compte Vancelian, le virement sera refusé ou gelé en attente de clarification. Vous devrez alors fournir une justification supplémentaire (document de mariage, procuration notariée, etc.) pour que la transaction soit débloquée — ce qui allonge considérablement le délai.
> 
> **Solution** : demandez à votre conjoint de faire un virement depuis son compte vers votre compte bancaire personnel d'abord. Une fois les fonds sur votre compte, vous pourrez effectuer un virement SEPA standard vers Vancelian sans friction.
> 
> Pour toute question sur votre situation spécifique ou si vous avez déjà soumis un virement dans cette situation, contactez le support Vancelian à support@vancelian.com — l'équipe pourra vous conseiller ou débloquer votre transaction si nécessaire.

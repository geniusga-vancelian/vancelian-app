# Comment je peux dépasser les limites de dépôt plus rapidement sans envoyer de documents ?

## Client question
> Comment je peux dépasser les limites de dépôt plus rapidement sans envoyer de documents ?

## Bot answer
> Je comprends votre impatience, mais il n'existe pas de raccourci pour contourner la vérification des limites de dépôt. Le processus de justification par document est obligatoire — c'est une exigence réglementaire, pas une formalité administrative que Vancelian peut contourner.
> 
> Pour augmenter vos limites de dépôt, vous devez soumettre un document justificatif via **Profil > Mon compte > Seuils de dépôt > Télécharger un document justificatif**. Les documents acceptés sont les mêmes que ceux pour la vérification de niveau 3 : fiche de paie (< 3 mois), dernier avis d'imposition, attestation de prestations sociales, preuve de patrimoine (héritage, vente de bien immobilier), relevé de crypto-gains, ou documents d'investissement, selon votre situation.
> 
> L'équipe de conformité examine chaque document manuellement — le traitement prend jusqu'à 3 jours ouvrés. Une fois approuvé, vos nouvelles limites s'appliquent immédiatement.
> 
> Si le bouton de téléchargement n'apparaît pas dans l'app, écrivez à support@vancelian.com avec le document en pièce jointe.

# Can I cancel an outgoing transfer?

## Short answer
Once an outgoing transfer has been initiated, it is no longer possible to cancel it, even if it appears as “in progress” or “pending” in your application. This status simply indicates that the transaction is currently being processed, but it does not allow for any modification or cancellation once.

## Details
Once an outgoing transfer has been initiated, it is no longer possible to cancel it, even if it appears as “**in progress**” or “**pending**” in your application.

This status simply indicates that the transaction is currently being processed, but it does not allow for any modification or cancellation once the transfer has been launched.

If the transfer was sent in error, we recommend **contacting the beneficiary** directly to try to recover the funds.

#### **Beneficiary verification before sending**

Before confirming a transfer, Vancelian performs a **beneficiary verification** to ensure that the name entered matches the corresponding bank account.

We strongly encourage users to use this feature to **reduce the risk of errors** or **fraud** before initiating a transfer.

[Learn more about beneficiary verification here.](https://support.vancelian.com/hc/en-gb/articles/39614974441361-What-is-beneficiary-verification)

# How can I change my PIN code to access the application?

## Short answer
You can change your PIN code directly from your Vancelian application by following the steps below. Access the Security and Privacy section from your Profile, select Change my PIN code, and enter your current PIN code followed by your new PIN code twice to confirm.

## Details
You can change your PIN code directly from your Vancelian application by following the steps below:

**Steps to change your PIN code**

- From the home page, open your **Profile**

- Go to the **Security and Privacy** section

- Select **Change my PIN code**

- Enter your **current PIN code**

- Then enter your **new PIN code twice** to confirm

- If the code is accepted, a **confirmation notification** will appear

**Important points to follow**

For security reasons, your new PIN code must not contain:

- A **logical sequence** of numbers (e.g. 123456, 456789),

- **Repetitive numbers** (e.g. 000000, 111111).

If the chosen code does not meet these criteria, an **error message** will prompt you to select another one.

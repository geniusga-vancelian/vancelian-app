# Why am I not receiving my confirmation code by SMS?

## Short answer

## Details
If you are not receiving the verification code, several reasons may explain it.

**1. Check your mobile network**

A network issue is the **most common **cause.

In this case, the message cannot be delivered and Vancelian cannot intervene.

Try moving to an area with a more **stable signal.**

**2. You are abroad**

When travelling internationally, some operators automatically** block** short SMS messages (such as OTP codes).

Contact your mobile **operator** to ensure that international SMS / OTP messages are not blocked.

It is also possible to request that the confirmation code be sent via **WhatsApp** as an alternative to SMS.

**3. Check your configuration (dual SIM)**

On some phones with two SIM cards, message delivery may be blocked if the wrong number is selected.

To check your configuration:

- Make sure **both SIM** cards are inserted.

- Go to **Settings → Dual SIM / SIM** **management.**

- Choose the SIM card used to receive SMS (SIM 1 or SIM 2).

**4. Too many attempts: sending temporarily blocked**

If you have attempted to receive a code too many times, the system may automatically block further requests.

In this case, you will need to wait **24 hours** before requesting a new code.

Please avoid repeated attempts. If the issue persists after 2–3 attempts, [contact our customer support](https://support.vancelian.com/hc/en-gb/articles/4409555298577-How-to-contact-Customer-Support)

 

#### **Using an authentication application**
If you experience recurring difficulties receiving SMS confirmation codes, you may choose to use two-factor authentication via an **authentication application (Authenticator).**

Once the Authenticator is set up on your account, certain operations can be validated using a code generated by the application, without relying on SMS delivery.

**To activate this option, the Authenticator application must be configured beforehand.**

The setup procedure is described in detail in the following article: [How do I set up my Authenticator](https://support.vancelian.com/hc/en-gb/articles/19685868023313-How-do-I-set-up-my-Authenticator)

For more information about account security and two-factor authentication, you can consult this article: [How does two-factor authentication (2FA) work on the Vancelian application?](https://support.vancelian.com/hc/en-gb/categories/10834688753809-Security-and-Troubleshooting)

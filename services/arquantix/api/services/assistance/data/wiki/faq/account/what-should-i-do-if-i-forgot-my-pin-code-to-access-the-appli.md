# What should I do if I forgot my PIN code to access the application?

## Short answer
If you forgot your PIN code, you can reset it directly from the login screen. How to reset your PIN code?

## Details
If you forgot your PIN code, you can reset it directly from the login screen.

**How to reset your PIN code?**

- Open the application and, on the screen where your PIN is requested, click **Forgot PIN** at the bottom of the page

- Choose a **new PIN code**

- Enter this new PIN **twice** to confirm

- Then enter the verification code received by **SMS**

- Your new PIN will be **active** immediately.

**Important points**

For security reasons, your PIN code must not contain:

- A **sequence of numbers** (e.g. 123456, 456789)

- **Repeated identical numbers **(e.g. 000000, 111111)

If you enter a non-compliant code, an error message will appear.

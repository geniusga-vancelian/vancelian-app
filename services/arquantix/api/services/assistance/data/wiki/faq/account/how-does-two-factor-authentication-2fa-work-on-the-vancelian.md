# How does two-factor authentication (2FA) work on the Vancelian application?

## Short answer
In order to strengthen account security, Vancelian offers two-factor authentication (2FA). This mechanism adds an additional verification step during certain sensitive actions in order to secure your operations.

## Details
In order to strengthen account security, Vancelian offers two-factor authentication (2FA).

This mechanism adds an additional verification step during certain sensitive actions in order to secure your operations.

**What types of 2FA are available?**

Two two-factor authentication methods are used in the Vancelian application:

- **The verification code sent by SMS**

- **The code generated by an authentication application (Authenticator)**

Depending on the operations performed and the configuration of your account, one or the other of these methods may be used.

As a reminder, the use of an Authenticator application is already mandatory for validating crypto-asset withdrawals. This requirement remains unchanged and forms part of the security measures implemented to protect users’ funds.

**Using 2FA via an Authenticator application**

Until now, certain operations, such as logging into the application or validating an outgoing transfer, required a code sent by SMS.

From now on, when authentication via an Authenticator application is enabled on your account, it is possible to validate certain operations using a code generated by your authentication application, instead of an SMS code.

**This option is only available if the Authenticator has been previously set up in the application.**

**How do I enable authentication via an Authenticator application?**

In order to use 2FA via an Authenticator application, it must first be configured on your Vancelian account.

The setup procedure is described in detail in the following article: [How do I set up my Authenticator](https://support.vancelian.com/hc/en-gb/articles/19685868023313-How-do-I-set-up-my-Authenticator)

Once the Authenticator has been correctly activated, it can be used to validate certain operations requiring enhanced authentication.

If you need to change your Authenticator application, please [contact our support team](https://support.vancelian.com/hc/en-gb/categories/29189853769361-About-Vancelian). An identity verification will be required so that we can remove the old key before you can register a new one.

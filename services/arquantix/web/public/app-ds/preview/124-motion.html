Motion · Webapp4

Discret, court, naturel. Trois durées, deux easings.

--v-motion-fast 120 ms Hover, focus, color shift
--v-motion-base 200 ms Section enter, fade-up
--v-motion-slow 320 ms Page transition, drawer
--v-ease-out cubic-bezier(0.22, 1, 0.36, 1) --v-ease-in-out cubic-bezier(0.65, 0, 0.35, 1)

Notes d'usage

Si tu hésites, prends --v-motion-fast + --v-ease-out. Animation = service, pas spectacle.

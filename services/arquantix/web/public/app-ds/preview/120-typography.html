Notes d'usage

Newsreader sur un bouton est un bug. Inter sur le wordmark est un bug. Le mix Inter + Newsreader fonctionne parce que chaque rôle est strict. Pour les montants, on utilise les classes .v-amount-* qui ajoutent tnum et lnum — pas de chiffres oldstyle accidentels.

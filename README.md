# vancelian-app

Application Vancelian (backend, frontend, agent IA).

## Arquantix — démarrage rapide

Prérequis : **Docker Desktop** + **Git**. Copier `.env.arquantix.example` → `.env.arquantix` si besoin.

```bash
make setup
```

Puis : Web <http://localhost:3000> · API <http://127.0.0.1:8000/docs>
